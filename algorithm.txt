В таблице в ячейках есть либо адреса ячеек, либо числа. Требуется вычислить значение произвольной заданной ячейки.
Вычисление значения ячейки заканчивается, когда мы получаем целое число. Вычисление может идти линейно, может ветвиться.

Для вычисления значения ячейки надо:
1. Получить таблицу. Вызвать csv::calculate для ячейки.
2. Убрать знак "=", если необходимо. Если значение первого символа в ячейке - не число, получить значение следующей
ячейки по адресу из текущей ячейки. Если значение следующей ячейки не число, вызвать csv::calculate, передав значение
следующей ячейки, иначе вернуть значение следующей ячейки.
3. Если значение первого символа в ячейке - число, вернуть значение ячейки.
4. Для случая выражения (ARG1 OP ARG2) надо:
4.1 Выявить, что это выражение по наличию арифметической операции из списка: +, -, *, /.
4.2 Выделить первый аргумент, второй аргумент, операцию. Вызвать csv::calculate для первого и второго аргумента.
4.3 После получения чисел вместо первого и второго аргумента, выполнить арифметическое действие, получившееся число
вернуть как результат выполнения функции.
